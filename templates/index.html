How it works

Always-on
AI Guardian.

RAKSHAK's threat engine sends silent heartbeat pings every 2 minutes. Miss three in a row? Auto-SOS fires — zero input needed.

PythonFlaskSocketIOTiDBAI/ML
rakshak_engine.py
# RAKSHAK AI Threat Engine v4.0
from rakshak import ThreatEngine
 
engine = ThreatEngine(mode="guardian")
engine.set_interval(seconds=120)
engine.set_threshold(missed=3)
 
# Auto-SOS after 3 missed pings
async def monitor(user):
  if user.missed_pings >= 3:
    await engine.deploy_sos(user)
    notify_contacts(user.emergency_list)
    broadcast_location(user.gps)
 
engine.start() # Shield active
